### 2. **р╕Ыр╕гр╕▒р╕Ър╕Ыр╕гр╕╕р╕Зр╕Вр╣Йр╕нр╕бр╕╣р╕е (Data Preprocessing)** ЁЯЦ╝я╕П

image_processor = AutoImageProcessor.from_pretrained("google/vit-base-patch16-224-in21k")

    images = [x.convert("RGB") for x in example_batch['image']]  # р╣Бр╕Ыр╕ер╕Зр╣Ар╕Ыр╣Зр╕Щ RGB
    inputs = image_processor(images, return_tensors='pt')  # р╕Ыр╕гр╕▒р╕Ър╕Вр╕Щр╕▓р╕Фр╣Бр╕ер╕░р╣Бр╕Ыр╕ер╕Зр╣Ар╕Ыр╣Зр╕Щ Tensor
